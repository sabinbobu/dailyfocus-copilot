# DailyFocus Copilot

A GenAI-powered personal planning assistant focused on **daily usefulness, grounded answers, and measurable quality**.

## Why this project
Most GenAI apps are demos. This one is designed to be used every day and evaluated like a real product.

## Features (Planned)
- Daily planning with citations
- Task prioritization using personal context
- Decision support with traceable reasoning
- Evaluation-first RAG

## Tech Stack
- FastAPI
- PostgreSQL + pgvector
- OpenAI API
- pytest, ruff, mypy
